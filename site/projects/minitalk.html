Minitalk

C

A client and server that transmit a message between two processes using only UNIX signals, one bit per signal.

View on GitHub

A small client and server pair that sends a text message between two processes using nothing but UNIX signals. The client turns each character into its 8 bits and fires one signal per bit, SIGUSR1 for a 0 and SIGUSR2 for a 1. The server catches those signals, rebuilds each byte bit by bit, and prints the message as it arrives.
